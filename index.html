MobileNet image classifier

This image classifier uses your webcam to predict whatever is in front of it.

Instructions

  1. Make sure your device's camera privacy settings allow the browser to use the webcam
  2. Place three different kinds of objects (A, B, C categories) in front of the camera
  3. First, train the model with a no objects in front of the camera, and click the 'No Action' button a few times
  4. Train the model by taking several pictures of each object type with the A, B and C buttons respectively
  5. Have fun showing new objects of the same three classes to your camera, see what happens when you show a new object
